## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

Link to the back end repo [back end repo](https://github.com/dieum512/hello-rails-back-end).

### Prerequisites

JavaScript
CSS

### Setup

$ git clone git@github.com:dieum512/hello-react-front-end.git

$ cd hello-rails-front-end

### Setup

Install all project dependencies by running the command below

$ npm install

### Run the app

$ npm start

### Run tests

To run tests, run the following command:

Example command:

```sh
  npm run tes
```
